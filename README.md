#  Human Flourishing Course Reflection Website

##  Overview
This website is a personal reflection on the five dimensions of human flourishing: **Foundations**, **Awareness**, **Connection**, **Wisdom**, and **Integration**. It was created for a non-CS course to visually organize key concepts, personal takeaways, and guided meditation resources.

---

##  Purpose
The site serves as a digital portfolio to:
- Summarize core topics from the Human Flourishing curriculum
- Share personal insights and growth strategies
- Provide access to meditation practices aligned with each theme

---

##  Technologies Used
- **HTML5** – Semantic structure and content layout
- **CSS3** – Styling, transitions, and responsive design
- **JavaScript** – Smooth scrolling for navigation links

---

##  Features
- Sticky navigation bar with smooth scrolling
- Thematic sections with topic blocks and takeaways
- Embedded links to guided meditations (YouTube or course files)
- Responsive design for desktop and mobile
- Hover effects and subtle transitions for interactivity

---

##  Responsive Design
The layout adapts for smaller screens:
- Navigation wraps for mobile
- Section padding and font sizes adjust for readability

---

## Meditation Resources
Each topic includes a curated meditation link to encourage deeper engagement with the material.

---

##  Author Notes
This project reflects my personal journey through the Human Flourishing course. It blends academic insight with practical tools for well-being and is designed to be both informative and meditative.
